WebAR kamera funksiyasi to'g'ri ishlashi uchun sayt HTTPS orqali ochilishi kerak. GitHub Pages, Netlify yoki Vercel mos keladi.

HTTPS server AR uchun muhim

Kamera ruxsati ko'p brauzerlarda faqat xavfsiz HTTPS manzilda barqaror ishlaydi.

GitHub Pages

GitHubda yangi repository yarating va loyiha fayllarini yuklang.

Settings bo'limidan Pages sahifasiga o'ting.

Branch sifatida main va papka sifatida /root ni tanlang.

Berilgan HTTPS linkni telefon brauzerida ochib, kamera ruxsatini bering.

Netlify yoki Vercel

Loyiha papkasini ZIPdan chiqaring.

Netlify yoki Vercel oynasiga papkani yuklang.

Avtomatik yaratilgan HTTPS manzil orqali saytni tekshiring.

Lokal sinov

Windowsda start_local_server.bat faylini ishga tushiring.

Yoki terminalda python -m http.server 8080 buyrug'ini bajaring.

Brauzerda http://localhost:8080 manzilini oching.

Muhim: AR kamera qismi to'liq ishlashi uchun HTTPS server tavsiya etiladi. Lokal sahifalar ochiladi, lekin ayrim mobil brauzerlar HTTPSsiz kameraga ruxsat bermasligi mumkin. AR sahifa HIRO marker bilan ishlaydi.
