# AR-платформа для профессионального обучения

Тема дипломного проекта: использование технологий дополненной реальности (AR) при преподавании профессиональных дисциплин.

Автор: Абдурахимов Сиротиллохужа Улугбек угли.

## Состав проекта

- `index.html` - главная страница и общая информация о проекте
- `pages/topics.html` - темы Router, Switch, Server, LAN topology, OSI model, Wi-Fi
- `pages/models.html` - библиотека 3D-моделей
- `pages/ar-view.html` - AR-просмотр через HIRO marker
- `pages/test.html` - тест для проверки знаний
- `pages/result.html` - страница результата теста
- `pages/deploy.html` - инструкция для GitHub Pages и локального сервера
- `start_local_server.bat` - запуск локального сервера в Windows

## Запуск

1. Распакуйте ZIP-файл.
2. В Windows запустите `start_local_server.bat`.
3. Откройте в браузере `http://localhost:8080`.

Запуск через терминал:

```bash
python -m http.server 8080
```

## Использование AR

1. Разместите проект на HTTPS-сервере: GitHub Pages, Netlify или Vercel.
2. Откройте сайт в браузере телефона.
3. Перейдите на страницу `AR ko'rish`.
4. Разрешите доступ к камере и наведите камеру на HIRO marker.

Важно: для полноценной работы AR-камеры рекомендуется HTTPS. Локальные страницы откроются, но некоторые мобильные браузеры могут блокировать камеру без HTTPS.
