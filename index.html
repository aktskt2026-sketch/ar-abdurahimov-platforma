Loyiha haqida

Ushbu WebAR o'quv platforma prototipi Abdurahimov Sirotilloxo'ja Ulug'bek o'g'lining "Kasbiy fanlarni o'qitishda kengaytirilgan haqiqat (AR) texnologiyalaridan foydalanish" mavzusiga moslashtirildi.

  • Kompyuter tarmoqlari bo'yicha ko'rgazmali o'quv materiallari jamlangan.
  • AR kamera bo'limi HTTPS serverda to'liq ishlashi uchun tayyorlangan.
  • GitHub Pages yoki lokal server orqali ishga tushirish yo'riqnomasi qo'shilgan.
